Elevation & Interaction

No drop shadows are used. Elevation is communicated through background-brightness overlays applied via CSS pseudo-elements. This works automatically in both light and dark mode without separate dark-mode overrides.

Overlay values

--elevate-1light: rgba(0,0,0,.03) · dark: rgba(255,255,255,.04)
--elevate-2light: rgba(0,0,0,.08) · dark: rgba(255,255,255,.09)

CSS utility classes

hover-elevate — level 1 on hover

hover-elevate-2 — level 2 on hover

active-elevate — level 1 on active/press

toggle-elevate — level 2 when toggled
